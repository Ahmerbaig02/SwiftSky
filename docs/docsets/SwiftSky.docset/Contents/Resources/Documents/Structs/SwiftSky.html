  • get(_:at:on:_:)

    Used to get a Forecast from the Dark Sky Servers

    Example Request

    SwiftSky.get([.current,.minutes,.hours,.days],
   at: Location(latitude: 0.0, longitude: 0.0)
) { result in
    switch result {
    case .success(let forecast):
        // do something with forecast
    case .failure(let error):
        // handle error
    }
}
